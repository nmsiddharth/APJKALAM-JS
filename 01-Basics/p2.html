        DATATYPE
        1) Primitive Data Types
        2) Non-Primitive Data Types

    1) PRIMITIVE Data Types

       - Int (Integer)-> It Accept only round numbers Ex = 0-9
       - Float -> It Accept only decimal value Ex = 10.5
       - Boolean -> Either True or False
       - Char -> It Accept Single Character Ex = A
       - Void -> Returns NULL value

    2) NON-PRIMITIVE Data Types

       - Array
       - Structure
       - Union
       - Enum
       - Pointer
       - Function
